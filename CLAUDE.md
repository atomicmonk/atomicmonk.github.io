# atomicmonk.github.io

Personal GitHub Pages site for Brian (atomicmonk).

## Site Structure

- `index.html` — Main landing page
- Static HTML, no build process required
- Deployed automatically from the `master` branch via GitHub Pages

## Development Workflow

- Work on feature branches (e.g. `feature/...` or `claude/...`)
- Merge to `master` to publish changes live
- No build step needed — edits to HTML/CSS/JS are deployed as-is

## Deployment

GitHub Pages publishes automatically from the `master` branch.
Live site: https://atomicmonk.github.io
